Convert a .csv text file to a .kml file for use with Google Earth or Google Maps.

Required file format: LocationName, Longitude, Latitude, Marker code (optional)
Values should be decimals (e.g. 45.234134) not dms (e.g. 45.23'18").
Marker codes: 1=green, 2=red, 3=purple, 4=yellow, blank or anything else=blue
